# Menú Web Sabore's

Sitio web responsivo para el menú del restaurante Sabore's con actualización de precios mediante JSON.

## Características

- ✅ Diseño responsivo de una sola columna
- ✅ Precios actualizables desde archivo JSON
- ✅ Animaciones y transiciones suaves
- ✅ Sección de métodos de pago
- ✅ Sistema de extras
- ✅ Simulador de actualización de precios
- ✅ Compatible con dispositivos móviles

## Estructura de Archivos
sabores-menu/
├── index.html # Página principal
├── style.css # Estilos CSS
├── script.js # Lógica JavaScript
├── precios.json # Archivo JSON con precios
└── menu-data.js # Datos estructurados del menú


## Instalación y Uso

1. Descarga todos los archivos en una carpeta
2. Abre `index.html` en tu navegador
3. Para actualizar precios, edita el archivo `precios.json`
4. Recarga la página para ver los cambios

## Actualización de Precios

Los precios se cargan desde `precios.json`. Para actualizar:

1. Abre `precios.json`
2. Modifica los valores de `precio`
3. Guarda el archivo
4. Recarga la página web

## Personalización

- Cambia colores en `:root` de `style.css`
- Modifica fuentes en las importaciones de Google Fonts
- Añade nuevas secciones editando `menu-data.js`
- Agrega métodos de pago en el HTML
